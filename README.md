##Pre-requiestes
Pgadmin4 up and running
Xammp MySql running

To start this app:
1. cd backend
2. npm install
3. node db_init.js //to create db and tables in Mysql DB;
4. node pg_init.js //to create db and tables in  Postgres DB;
5. node server.js //to start backend;
6. cd ..
7. npm install
8. npx expo start
